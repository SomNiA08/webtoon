## 4. 플롯·관계·정보 상태 (EP01 v2 종료 시점 스냅샷)

| 항목 | 상태 |
|---|---|
| **HANJI** | 자기가 무심코 한 것(*"여기 시야가 없었어요"*)에 **직업 이름이 붙는 것을 경험했다**(전력분석관). 마지막에 아카데미 명패를 **말없이 올려다본다**(P054, 무대사). **아직 진학을 결정하지 않았다.** 부모에게 아무 말도 하지 않았다 |
| **RAKU** | 하나(*"들어가기 전에 시야를 봐"*)를 배워 **한 판 안에서 결과가 바뀌는 것을 경험했다**. 완전히 굽히진 않았다(*"…한 번 더 해볼게요."*) 마지막에 **HANJI에게 배우려 묻는다**(*"형, 이거 어떻게 봐요?"*) |
| **HOO** | 두 사람에게 **각각 다른 답**을 줬다. RAKU에게 *"혼자 하면 딱 지금까지야."* / HANJI에게 *"프로는 안 될 수도 있어. 근데 네가 뭘 잘하는지는 여기서 알아볼 수 있어."* |
| **관계** | RAKU → HANJI: **깔봄 → 배움**(호칭 "형"의 온도 반전 완료). HANJI ↔ RAKU: 아직 서로를 잘 모른다. HOO는 둘 다에게 멘토 |
| **정보 비대칭** | **HANJI의 부모는 아무것도 모른다** — 화면에 등장하지 않았고, HANJI가 오늘 무엇을 알게 됐는지 모른다. **이것이 EP02의 엔진이다** |
| **중심 갈등(연재 엔진)** | *"게임을 잘한다"의 정의* — RAKU=**손**(실력) / HANJI=**눈**(판단). **EP01에서 해소되지 않았다.** 성장 엔진으로 남는다 |

## 6. EP02 시드 (사용자 승인 · continuity-manager ↔ showrunner 합의)

> **게임을 전혀 모르는 학부모 동물 캐릭터**를 신규 투입해, ***"게임으로 뭘 먹고사냐"*** 는 반대(W1)를 **정면으로** 다룬다.

**이어받기 포인트**: EP01 P054 — HANJI가 명패를 올려다본 그 순간의 **직후**. 그가 집에 돌아가 부모와 마주치는 지점이 EP02의 문이다.

**⚠️ 제작 착수 전 선행 조건 (차단 조건)**
1. **학부모는 신규 캐릭터다 — 사용자 원화를 받거나 디자인 승인을 먼저 받아야 한다.** 하네스는 캐릭터를 창작하지 않는다(refs/ SSOT 규약). **원화 없이 렌더 착수 금지.**
2. **`LOC_GAMESCREEN` 고정 토큰 신설**(§2 · `retro_ep01.md` §4-1) — 프롬프트 작성 전.
3. **사실 벽 §3 유지** — 부모의 반대를 다루면서도 수강료·기간·실적·취업률로 반박하지 않는다. **미확인 항목으로 설득하면 거짓 광고다.** 반박의 재료는 **EP01에서 이미 확보된 것**(직업이 실재한다 · 배우면 는다)뿐이다.

**주의사항 체크리스트 (명령형)**
- [ ] HANJI=청록 / RAKU=빨강+테이프 / HOO=네이비 헤드셋 — 색을 바꾸지 마라
- [ ] RAKU만 초록 눈
- [ ] HOO를 **강사·코치로 부르지 마라** (마스코트 멘토)
- [ ] 배경 인물도 의인화 동물 (`no human hair`)
- [ ] 화면 문자는 **화이트리스트**(대본 자구 + 지정 라벨)만 — 그 밖 문자 = REGEN
- [ ] 무표정 = REGEN (표정 연기 필수)
- [ ] **W1을 회수하라** — 심은 복선은 회수 의무가 있다
