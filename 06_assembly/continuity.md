## 2. 설정·세계관 원장

> **정본**: `world.md`. EP01에서 실제 확립/노출된 설정만 기입. 다음 회차는 이와 모순 없이 이어간다.

### 2-A. 게임 《아크라이트(ARKLIGHT)》 규칙 — EP01 확립분

- **종목**: 5v5 실시간 팀 전략전. 상대 진영 심장부 **「코어(Core)」** 파괴 = 승리.
- **맵 구조(EP01 미니맵 SSOT로 시각 확정)**: 3공격로 **레일(RAIL) / 스파인(SPINE) / 서지(SURGE)** + 중앙 자원지대 **러스트필드(RUSTFIELD)** + 중립 대형 오브젝트 **타이탄(TITAN, 레드 표기)**. → P03·P36·P44·P49 미니맵에서 이 배치로 고정 렌더됨. **다음 회차 미니맵도 이 레이아웃 유지.**
- **포지션 5종**: 밴가드(태산)·레이서(소예, EP04)·아크메이지(도경)·서포트(하루)·**콜러(유건=사령탑, 진짜 무기)**.
- **콜 오더 = 인게임 사령탑의 지시.** 유건의 콜러 두뇌가 힘숨찐 정합의 핵심.

### 2-B. 아카데미 반 구조 — EP01 확립분

- **테라 e스포츠 아카데미**: 프로 지망생 사관학교. 반 = 실력 서열 리그(월말 승강전).
- **반 서열(위→아래)**: 1군 **크라운 룸**(딥네이비+골드) / 2군 메인 스테이지(차콜+크림슨) / 3군 그라인더(올리브+브론즈) / **4군 콜드 벤치(회철색+청록 트림) = 유건 시작점.**
- **유니폼 색 SSOT**(characters §1): EP01 렌더 확인 — 유건/콜드벤치 = 회철+청록, 윤태오/1군 = 네이비+골드. 다음 회차 반별 색 고정.
- **티어 시스템**: 슬래그(Slag)→코퍼→브라스→크롬→티타늄→미스릴→플라즈마→노바→싱귤래리티. 유건 = 최하위 **슬래그(SLAG)** 배지(P23, 탁한 회철). 성장 UI = 헌터물 시스템창 톤.
- **MMR/AAS**: 숨은 점수(반 배정·조작 표적). EP01 P43에서 도경이 "승률 12%" 통계 인지 — **W5(조작) 미세 단서 최초 노출.**

### 2-C. desync 규칙(회귀 엔진) — EP01 상태 = **셋업 단계**

> **핵심: EP01은 desync 규칙을 심기만 하고 실전 발현은 없다.** 잔상 예지가 아직 "유효(선명)"한 상태.

- **회귀 물리**: 「크라운」결승 파멸 → 7년 전 입학 첫날로 의식만 회귀. 지식·감각 보존, 육체 리셋(손 굳음·트라우마 떨림).
- **잔상(Afterimage)**: 유건 전용 청록 예지 오버레이. EP01에서 **유효=선명 상태로만 작동**(P03 결승 시점 동기화, P34 힘숨찐 예지 선명). 붉은 글리치=단 1회 P05(결승 붕괴 순간 어긋남), 역재생 파편화=P13(회귀 트리거).
- **desync 법칙 발현 로드맵(다음 회차 예고)**:
  - **EP04 = 첫 desync 발현**(잔상 첫 붉은 찢김). world §1-B 법칙1(관찰-반응 임계) 발동 예정.
  - **P57 "이번엔 이긴다" = desync 점화 씨앗**(G3 과신 개입의 심리적 뇌관). EP04 승강전 도전 선언으로 회수.
  - EP06 = 미래 완전 붕괴("이제부터 나도 모른다"), EP11 = desync 역이용(저주→축복).
- **정합성 게이트**: 모든 "빗나간 미래" 반전은 desync 법칙1~3으로 설명 가능해야 함(world §1-E). 다음 회차 반전 설계 시 twist-master 필수 준수.

### 2-D. 세력·적대 구조 — EP01 노출 상태

- **가시적 적대 = 고형준 총감독만**(P31 복도 원경, 얼굴/대사 없음). EP02에서 "30일 뒤 승강전 못 이기면 콜드 벤치 해체" 통보로 본격 등장 예정.

## 5. 다음 회차(EP02) 주의사항 【일관성·회수 체크리스트 — 명령형】

### 5-A. 이어받기 지점 (클리프행어 연결)

- **EP01 종료 = P57 직후.** EP02는 **윤태오의 "…그 계정, 너였어?"(P54) 직후 상태**에서 출발한다. 유건의 "이번엔 이긴다"(P57) 결의가 깔린 채, 정체가 윤태오에게 인지된 위기로 문을 연다.
- **정보 상태 인수**: 윤태오 = "부계정=ORACLE" 확신 / 유건 = 아직 들킨 걸 모름(잔상 흔들림으로 독자만 시인 목격). EP02는 이 비대칭에서 **부인전 + 윤태오 스스로 확신 접기("설마, 이런 꼴찌 반에?")**로 정체를 재봉인(twist-plan §2 EP02).

### 5-B. 외형 일관성 (반드시 대조)

1. **refs/ 7종 재사용**(재렌더 금지). EP02 등장 캐릭터(유건·윤태오·태산·하루·한상혁 확정 + 고형준 본격 등장)는 refs PNG를 앵커로.
2. **유건 오른손등 흉터** 위치 고정(손 등장 시). **헤드셋·회철+청록 트랙탑** 유지.
3. **정하루 곱슬·왼볼 보조개·매니저 완장 강조 렌더**(EP01 P32 남성적 경향 재발 방지).
4. **한상혁 오른손 관절 보호대** 유지(W4 시각 복선 — EP02에서 이중대사와 함께 강조).
5. **잔상 = 유건 전용 청록** 규약 유지. 색온도 = 기본 청록/회철, 온난색은 각성 컷만.

### 5-C. 설정 연속성 (모순 방지)

1. **콜 오더 SSOT "서지 셋. 타이탄 버려."** — EP02에서 오더 재현·인용 시 **자구 그대로 베이크**(반전 공정성 자산). 미니맵 레이아웃(RAIL/SPINE/SURGE/RUSTFIELD/TITAN) 유지.
2. **로스터 = 태산·하루·도경 3인**(소예 미등장 — EP04 영입 전까지 호명 금지). EP01 P37에서 교정한 오류("소예 자리 지켜" 삭제)를 EP02도 준수.
3. **desync 미발현 유지** — EP02는 아직 잔상 유효(선명). 첫 붉은 찢김은 EP04. EP02에서 잔상을 붉게 그리지 말 것.
4. **딜러/핸드 미등장 유지**(W3 은닉). 적대는 고형준까지만.

### 5-D. 떡밥 회수 감시 (twist-master·series-plotter 인계)

1. **카드A 재봉인 필수** — 윤태오 인지를 EP02에서 "가짜 안도"로 유예(방치 시 반전 소모). 단, 경계는 무의식에 남겨 EP08 재점화 관성 확보.
2. **W4 첫 이중대사 심기** — 한상혁 "나 같은 놈은 믿지 마라. 코치란 건, 언젠간 선수를 파는 자리니까."(EP02 신규 심기 예정). 지금은 겸손한 자조로 위장.
3. **해체 데드라인 점화** — 고형준 "30일 뒤 승강전 못 이기면 콜드 벤치는 없어진다"(타이머 = 정체 유예를 대체하는 더 큰 위협).
4. **W1·W2 절대 회수 금지**(엔진 근원). EP02는 W1 심화(재노출)만 허용, 답 금지.

### 5-E. 미착수 자산 준비 (EP04 대비 선행)
