> **설계 3원칙 (twist-patterns §1 준수)**
> 1. **사후 필연성** — 반전 후 되짚으면 단서가 보인다. 단서 없는 반전은 사기.
> 2. **시선 통제** — 진실은 숨기지 않고 "보이되 의미를 가린 채" 둔다. 독자 시선은 미스디렉션에 묶는다.
> 3. **대가 발생** — 모든 반전은 캐릭터에게 새 딜레마/비용을 남겨 다음 회차 엔진이 된다.
>
> **정합성 게이트 (world §1-E 필수)**: 모든 "빗나간 미래" 반전은 desync 법칙 1(관찰-반응 임계)·2(근접 감쇠)·3(고정점 관성) 중 하나로 **왜 어긋났는지 설명 가능**해야 통과. 근거 없는 "사실 미래가 달랐다"는 설정 붕괴.

## 0. 반전 엔진 요약 — 이 작품의 반전은 "정보 우위 ↔ desync"에서 나온다

이 작품은 반전 광산을 **두 축**에서 판다.

| 축 | 반전 성격 | 공정성(단서 짝) 근거 |
|---|---|---|
| **정체 축 (힘숨찐 개방)** | 유건의 3중 은폐(회귀/본실력/부계정 ORACLE)가 단계적으로 새어나가며 터지는 반전 | 유건의 프로급 콜·잔상·부계정 흔적을 "허세/우연/컨디션"으로 미스디렉션 후 회수 |
| **desync 축 (예지 붕괴)** | 유건이 "안다고 믿은 미래"가 개입 때문에 어긋나는 반전 | 잔상 UI(청록 선명→붉은 글리치→파편화)가 desync를 **독자 눈앞에 미리** 보여주는 시각 복선 |

두 축이 톱니처럼 맞물린다: **정체를 흘려 이길수록(정체 축) → 미래가 어긋난다(desync 축)**. 즉 사이다 반전이 곧 긴장 반전을 낳는 자가발전 구조. 그래서 "매 회차 반전 ≥1"이 소모되지 않고 재충전된다.

**미회수 복선 금지 원칙**: 심은 모든 복선은 §6 추적표에 회수 회차를 못박는다. 시즌1 미회수분(W1·W2·W3·W4·W6)은 EP12에 이월 신호를 심어 "빚의 이자"를 명시한다.

## 1. EP01 상세 반전 설계 【최우선 — 카드 A + 카드 C 정밀 심기/터뜨리기】

> EP01은 **2개의 반전**을 싣는다. ①클라이맥스 = 카드A(정체 반전, "…그 계정, 너였어?") ②2차 훅 = 카드C(시점 반전, "이번엔 이긴다"). 두 반전이 서로를 강화한다: C가 "저 무명이 곧 저 챔피언"임을 독자에게 확정시키는 순간, A의 "윤태오가 그걸 눈치챘다"가 공포로 완성된다.

### 1-A. EP01 반전 ① — 카드A "정체 반전" (클라이맥스 클리프행어)

| 항목 | 내용 |
|---|---|
| **반전 유형** | 정체 반전(twist-patterns 2.1) + 실력 반전(힘숨찐 개방, 2.10 약자⇄강자) |
| **독자 예측 (뒤집을 것)** | "이 무명 슬래그 꼴찌는 그냥 콜드 벤치에 처박힌 루저다. 실력의 편린은 어쩌다 나온 뽀록/허세다." |
| **뒤집기 (터뜨림)** | 1군 골든보이 윤태오가 유건의 콜 시그니처를 **전설의 계정 ORACLE**과 대조 → 확신. 무명의 정체가 "곧 정체를 들킬 前 세계 챔피언"으로 전복. |
| **터뜨리는 컷 (트리거)** | 긴 세로 여백 → 윤태오 클로즈업 → **대사만 있는 컷**: "…그 계정, 너였어?" (hook-analyst §3 결정적 대사 컷 = 대사 위주 톤 최적) |
| **충격 증폭** | 침묵 컷(여백) → 윤태오 눈 클로즈업 → 유건의 잔상이 미세하게 흔들리는 반응 컷(들켰다는 무언의 시인) |
| **desync 정합** | 이 반전은 desync가 아닌 **정체 축**. 단, 이 인지가 **G2 위반의 방아쇠** → EP02부터 1군·핸드 경계로 desync 가속(대가). |

**EP01 내 카드A 복선 심는 패널 위치 (3회 노출 법칙 — twist-patterns §3)**

| 노출 | 심는 위치 (series-arc §5-B 비트 기준) | 심는 방식 | 미스디렉션(오해 라벨) |
|---|---|---|---|
| **①무심** | **콜드 오픈 (비트1, ~5%)** 「크라운」결승. ORACLE의 오더 시퀀스가 화면에 흐른다. 스코어보드·중계에 "ORACLE" 닉네임 노출. | 독자는 이걸 "몰락하는 챔피언의 정보"로만 읽음. 유건과 연결 안 시킴(시점 전환으로 차단). | "이건 오프닝 도입부 딴 인물 이야기" |
| **②변주** | **힘숨찐 편린 (비트6, ~50~60%)** 첫 스크림/평가에서 유건이 콜 한 번. 그 오더의 **형태·리듬이 콜드 오픈 ORACLE 오더와 동일**(재독 시 매칭되는 시각 반복). 배경 UI에 부계정 로그인 흔적/닉 잔재가 1컷 스침. | "찍었어." 허세로 위장(hook-analyst 미스디렉션 라벨). UI 닉은 "지나가는 화면"으로 위장. | "무명이 어쩌다 한 번 잘 읽은 것 + 허세" |
| **③회수** | **클라이맥스 (비트7→9, ~80~95%)** 윤태오가 그 콜 시그니처에서 멈칫(비트7 "이 오더… 어디서 봤지?") → 부계정 흔적/콜 패턴을 ORACLE과 대조(비트8 빌드업) → 확신 대사(비트9). | — (여기서 미스디렉션 해제) | — |

> **공정성 검증**: 콜 오더의 시각적 형태를 콜드 오픈(①)과 힘숨찐 편린(②)에서 **동일하게** 그려두면, 독자는 재독 시 "아 저 오더가 그 오더였어!"를 무릎 친다. 윤태오가 오직 그만 알아보는 근거 = characters §4-1 "ORACLE 광신도, VOD·콜 패턴 통째 암기"로 이미 확보(날조 아님).

### 1-B. EP01 반전 ② — 카드C "시점 반전" (2차 훅)

| 항목 | 내용 |
|---|---|
| **반전 유형** | 시점 반전(twist-patterns 2.4 Unreliable) + 시간/기억 반전(2.3 회귀) |
| **독자 예측 (뒤집을 것)** | "콜드 오픈의 결승 붕괴 = 현재 진행형 사건이거나 그냥 배경 프롤로그다." |
| **뒤집기 (터뜨림)** | 도입 결승 붕괴는 **회귀 전 미래(=이미 겪은 파멸)**였고, 지금 입학일이 회귀 후 현재임이 드러남. 같은 장면이 "앞으로 막아야 할 파국"으로 의미 전복(재맥락화). |
| **터뜨리는 컷 (트리거)** | 클라이맥스 직후 유건 내적 선언 컷: **"이번엔… 이긴다."** (도입 붕괴 장면 인서트 1컷 재편집으로 의미 뒤집기 — twist-patterns 2.4 "재독 컷") |
| **충격 증폭** | 콜드 오픈의 손 떨림 컷과 현재의 굳지 않은 손 컷을 **대구(對句)**로 배치 → 독자가 "시간이 되감겼다"를 시각으로 확정. |
| **desync 정합** | 시점 반전 자체는 desync 무관(회귀 사실 자체). 단 이 선언이 **G3(과신 개입)의 씨앗** → EP04에서 첫 desync 발현으로 회수. |

**EP01 내 카드C 복선 심는 패널 위치**

| 노출 | 심는 위치 | 심는 방식 | 미스디렉션 |
|---|---|---|---|
| **①무심** | **회귀 트리거 (비트2, ~10%)** 파멸 순간 이질적 감각 — 잔상이 산산조각 → **역재생**되는 1컷. | "붕괴의 시각 연출"로 위장. 되감김임을 명시 안 함. | "패배의 절망적 이펙트" |
| **②변주** | **기상 (비트3, ~15%)** 7년 전 입학 첫날 아침. 유건이 **굳지 않은 손**을 응시. 속(서브텍스트): "…돌아왔어." | 독자는 "돌아왔어"의 의미를 아직 반쯤만 이해(회귀 사실은 독자에게 서서히). | "무슨 소리지? 데자뷰?" |
| **③회수** | **2차 훅 (비트10, ~95%)** 도입 붕괴=미래였음 확정 + "이번엔 이긴다". | — | — |

### 1-C. EP01 장기 우물 심기 (W1·W2 — 회수 금지·암시만)

> EP01은 회귀 엔진의 근원 미스터리 W1·W2를 **심되 절대 설명하지 않는다**(world §6: 성급 회수 시 엔진 정지). 이것이 카드C 위에 얹히는 "더 깊은 떡밥".

| 우물 | EP01 심는 패널 | 심는 방식 | 회수 예정 |
|---|---|---|---|
| **W1 (왜 회귀했나)** | 비트2 파멸 순간의 이질적 감각. 되감김이 **자연스럽지 않다**는 1컷(누가 손댄 듯한 위화감). | 붕괴 연출에 묻음. 원인 언급 0. | 시즌1 후반~시즌2 (🔒장기보류) |
| **W2 (잔상=기억인가 외부신호인가)** | 비트2 잔상의 **존재 자체** + 비트6 힘숨찐에서 잔상이 상대 무브를 비추는 첫 노출. | "회귀 능력의 당연한 사양"으로 위장. 잔상이 뭔지 묻지 않게. | 시즌1 중반 EP06~ (🔄진행) |

> **twist-master 셀프 게이트**: EP01에서 W1·W2에 답을 조금이라도 주면 안 된다. 잔상은 "그냥 그가 미래를 보는 장치"로만 기능. 이 장치가 "외부 신호일 수도"라는 의심은 EP05 미세 이상 컷(잔상이 유건도 모르는 것을 비춤)부터 시작.

### 1-D. EP01 반전이 남기는 대가 (다음 회차 엔진)

- **카드A의 대가**: 정체를 들킴 → **G2 위반 방아쇠 점화**. EP02는 "부인전 + 윤태오 재확신 vs 의심"의 미스디렉션 재봉인으로 시작. 1군·핸드의 경계가 앞당겨져 desync 가속의 씨앗.
- **카드C의 대가**: "이번엔 이긴다"는 통제 선언 → **G3 과신 개입의 심리적 뇌관**. EP04에서 승강전 도전 선언(첫 큰 개입) → 첫 desync 발현으로 회수. "이기려는 의지 자체가 미래를 부순다"는 아이러니 예약.

## 2. 회차별 반전 계획 (EP02~12) — 독자 예측 → 뒤집기 → 복선 위치 → 대가

> 각 회차 **최소 1개 반전 보장**. 큰 반전(회차 끝)+가능 시 중반 작은 반전. 모든 desync 반전에 §1-B 법칙 태그.

### EP02 — 정체 여파·부인전 / 강등(해체) 통보

| 항목 | 내용 |
|---|---|
| **반전 유형** | 정체 미스디렉션 재봉인(2.1 변주) — "가짜 안도"를 주는 역방향 반전 |
| **독자 예측** | "EP01에서 들켰으니 이제 정체가 폭로되고 유건이 위기다." |
| **뒤집기** | 윤태오가 **스스로 확신을 접는다** — "설마. ORACLE이 이런 꼴찌 반에? 헛것 봤나." (자기 우상을 이 밑바닥과 연결 못 하는 오만이 오히려 유건을 보호) → 폭로가 **유예**되며 긴장이 수면 아래로 잠복. |
| **작은 반전(중반)** | 콜드 벤치 첫 대면. 태산이 대들지만, 유건의 무심한 한마디가 태산의 강등 트라우마 정곡을 찌름 → "이 무명이 나를 읽었다"는 첫 균열. |
| **복선 심기** | **W4 첫 노출(무심)**: 한상혁의 이중 의미 대사 "나 같은 놈은 믿지 마라. 코치란 건, 언젠간 선수를 파는 자리니까." (지금은 겸손한 자조 / 반전 후엔 배신 고백). 패널 위치: 한상혁 첫 등장·유건에게 툭 던지는 컷. |
| **desync 정합** | 해당 없음(정체 축). 단 윤태오 경계가 무의식에 남아 EP08 재점화의 관성 확보(법칙2 근접감쇠: 유건이 계속 태오 근처에 있으므로 잠복 단서가 썩지 않음). |
| **대가** | 고형준 "30일 뒤 승강전 못 이기면 콜드 벤치는 없어진다" → **타이머 점화**. 정체 유예 = 안도지만, 해체 데드라인이라는 더 큰 위협으로 교체(상승 나선). |

### EP03 — 첫 비공식 스크림 / 힘숨찐 개방 / 도경의 데이터 의심

| 항목 | 내용 |
|---|---|
| **반전 유형** | 실력 반전(힘숨찐 개방, 2.10) + 정체 추적 서브플롯 개시(2.1 역방향 — 아군이 탐정) |
| **독자 예측** | "무명이 콜드 벤치 오합지졸로 뭘 하겠어. 태산도 안 믿잖아." |
| **뒤집기** | 유건의 콜 정확도가 스크림에서 폭발 → 태산이 **가장 먼저 진심으로 무릎 꿇는다**(첫 신뢰). 힘숨찐 1차 사이다 개방. |
| **작은 반전(회차 끝)** | 사이다 직후 **찬물**: 남도경이 데이터로 유건을 캐기 시작 — "이 콜… 승률 12%짜리예요. 통계적으로 불가능한데. 당신, 누구예요?" (아군이 독자 대리 의심자로 전환 = 정체 축 재점화) |

### EP11 — 클라이맥스 2부 / desync 역이용으로 조작 전복 / 각성 【국소 정점 10】

| 항목 | 내용 |

| **반전 유형** | **떡밥 회수 반전(2.5 Chekhov) + 규칙 반전의 재전복(2.7)** — 약점이 무기로 |

## 6. 팀 통신 (SendMessage 요지)

- **tension-engineer**: 반전 타이밍 = 긴장 정점과 일치. 큰 반전은 회차 끝(긴장 7~10), 작은 반전은 중반. 국소 정점 EP04·06·11에 desync 붕괴/역이용 반전을 배치했으니 비트 곡선의 정점을 이 컷에 맞춰라. EP01은 클라이맥스(카드A)+2차훅(카드C) **더블 피크**이므로 마지막 20%에 긴장을 두 번 치솟게 설계 필요.
- **series-plotter**: 떡밥 회수 영향 — W5를 EP11 부분회수로 확정(시즌1 카타르시스), W1·W2·W6·W4는 EP12 이월 신호로 정산. §3 추적표가 §6 트래커와 정합함을 확인. G4(하루)·자매 축을 시즌2 딜레마로 이월 표기 추가 권장.
- **episode-outliner**: EP01 §1의 3회 노출 패널 위치(콜드오픈 ORACLE 오더 = 힘숨찐 편린 오더 = 클라이맥스 대조)를 **시각적으로 동일한 오더 시퀀스**로 그려야 공정성 성립. 콜 오더 형태를 SSOT로 고정.
- **dialogue-writer**: 이중 의미 대사 필수 구현 — 한상혁 "코치란 건 언젠간 선수를 파는 자리니까"(EP02, W4), 윤태오 "그 계정, 너였어?"(EP01), 유건 "이번엔 이긴다"(EP01)·"이제부터 나도 모른다"(EP06). 재독 시 의미가 뒤집히도록 서브텍스트 설계.
