# 관찰 프레임 — 후킹·클리프행어·반전·반응 역설계 체크리스트

인기작을 부품 단위로 분해해 재현 가능한 패턴으로 되돌리기 위한 관찰 도구다. 사례가 아니라 **메커니즘**을 적는다("무엇"만큼 "어디서·어떻게"를 명시). 모든 관찰에 출처·작품·관측 시점을 붙인다.

---

## A. 1화 후킹 분해 체크리스트

첫 컷~첫 회차에서 독자를 낚는 장치를 순서대로 기록한다.

- **첫 컷 유형**: 충격 오프닝 / 강렬한 의문 / 즉시 위기 / 비범 선언 / 회귀·각성 트리거 / 일상 후 균열
- **의문 제기 시점**: 몇 번째 컷에서 "왜?"를 심는가
- **위기 도입 속도**: 위기가 첫 회차 어디쯤 등장하는가(빠를수록 이탈↓ 가설)
- **주인공 매력 제시**: 능력·결핍·목표 중 무엇을, 언제 보여주는가
- **세계관 노출량**: 1화에서 설명을 얼마나 절제하는가(과설명은 이탈 신호)
- **1화 끝 처리**: 어떤 미해결/반전으로 다음화를 강제하는가
- **작동 원리 한 줄**: 이 후킹이 자극하는 독자 욕구는 무엇인가

## B. 클리프행어 분해 체크리스트

회차 끝에서 긴장을 끊는 위치·방식을 기록한다.

- **유형**: 미해결 위기 / 충격 등장(새 인물) / 결정적 대사 / 정보 공개 직전 컷 / 반전 직격 / 관계 급변
- **위치**: 마지막 컷인가, 마지막 2~3컷의 빌드업이 있는가
- **정보 통제**: 무엇을 보여주고 무엇을 가렸는가(가린 것이 다음화 동력)
- **유도 강도**: "다음화"를 누르게 하는 압력이 약/중/강 중 어디인가
- **결제 구간과의 관계**: 유료 분기 직전 회차의 클리프행어가 더 강한가

## C. 반전(트위스트) 역설계 체크리스트

독자의 예측을 뒤집는 구조를 분해한다. **반전은 복선과 짝으로** 분석한다.

- **반전 유형**: 정체 반전 / 배신 / 관계 역전 / 시점 반전 / 진실 공개 / 능력·규칙 반전
- **뒤집힌 예측**: 독자가 믿고 있던 것은 무엇이었나
- **미스디렉션**: 어떤 장치로 독자를 오독하게 유도했나
- **복선 위치**: 단서를 앞 어느 패널/회차에 심었나("심기→숨기기→터뜨리기" 간격)
- **공정성 판정**: 앞에 단서가 있는 공정한 반전인가, 단서 없는 "날림 반전"인가
- **충격 증폭 장치**: 타이밍·컷 전환·대사·침묵 컷 중 무엇으로 충격을 키웠나
- **재사용 패턴 카드화**: 언제 / 어디에 / 어떻게 — 회차 설계에 이식 가능한 형태로 요약

## D. 복선 추적 표

| 단서(심기) | 위치(회차/패널) | 미스디렉션 | 회수(터뜨리기) | 간격 | 공정성 |
|---|---|---|---|---|---|
| 예) 주인공이 흘린 말 | 1화 후반 | 농담처럼 처리 | 12화 정체 반전 | 11화 | 공정 |

## E. 댓글 반응 코딩 표

베스트 댓글을 유형으로 집계한다. 개별 폭언이 아닌 **반복 패턴**으로 본다.

| 반응 유형 | 신호 예시 | 빈도(상/중/하) | 함의(기획 시사점) |
|---|---|---|---|
| 사이다 환호 | "통쾌하다", "참교육" | | 해당 전개를 강화·반복 |
| 고구마 불만 | "답답", "언제 터지냐" | | 누적 시 이탈 위험, 사이다 타이밍 조정 |
| 떡밥 추리 | "이거 복선 아님?" | | 복선 작동 신호, 회수 기대 관리 |
| 캐릭터 호불호 | "○○ 좋다/싫다" | | 캐릭터 비중·아크 조정 |
| 결제 불만 | "여기서 유료?" | | 결제 컷오프 위치 재검토 |

## F. 라이프사이클 판정 가이드 (트로프/장르)

- **상승**: 신작·상위권 진입 증가, 댓글에 신선함 언급, 모방작 등장 초기
- **정점**: 상위권 다수 점유, 변주작 포화, "이제 흔하다" 언급 시작
- **하락**: "또 이거냐" 피로 댓글, 신작 성적 저하, 변주 없는 답습

각 트로프 옆에 단계와 근거(관측 신호·출처)를 적는다. 단계는 시점 의존적이므로 관측 일자를 함께 남긴다.
