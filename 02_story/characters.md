## 0. 이 문서를 쓰는 사람에게 (v1 폐기 사유 승계)

v1은 캐릭터 두 가지로 실패했다:

1. **"표정이 없다"** — *"과장 리액션 금지·표정 변화는 작게"*가 하드룰이었다. 사용자가 만든 마스코트의 최대 자산(표정)을 스스로 봉인했다. **이 룰은 폐기됐다.**
2. **"얼굴이 안 보인다"** — 발·손·뒷모습 클로즈업이 절반 가까이. 캐릭터가 등장하지 않았다.

**v2의 기본값: 밝고, 표정이 풍부하다.** 놀람·억울함·뿌듯함·머쓱함이 **얼굴에** 보인다.
아래 §표정 레퍼토리는 prompt-smith가 **ACTING 블록으로 그대로 옮겨 쓰라고** 쓴 것이다 — 눈꺼풀·눈썹·부리/입·고개 각도 단위로 지정했다.

### 캐스트 벽 (⛔)
- **새 캐릭터를 만들지 마라.** 조연이 필요하면 실루엣 · 뒤통수 군중 · 화면 밖 목소리(off-panel).
- **HOO는 가상의 마스코트 멘토다.** 실존 강사로 묘사·암시 금지 (사실 벽 — brief §미확인).
- **HANJI의 부모는 언급만 된다.** 화면에 등장하지 않는다(회상 컷도 금지 — EP02의 씨앗이다).
- **헤드셋 색이 1차 식별자**: HOO=네이비 / HANJI=청록 / RAKU=빨강. **눈은 RAKU만 초록.**

## 1. HANJI (한지) — 펭귄 · 고3 · **주인공**

### 역할
이 회차의 반전이 일어나는 사람. **독자가 그와 함께 깨닫는다.**

| 항목 | 내용 |
|---|---|
| **신분·나이** | **고3.** 이스포츠 진학·진로를 고민 중. 아카데미 체험 수업에 **혼자, 자기 발로** 왔다 |
| **want (원하는 것)** | 이스포츠를 놓지 않아도 되는 **명분**. "이거 해도 된다"고 누가 말해주길 |
| **need (진짜 필요한 것)** | 명분이 아니라 **이름**. 자기가 이미 하고 있던 것이 **직업**이라는 사실 |
| **결함** | **자기 검열.** 답을 알고도 먼저 말하지 않는다. 말끝을 흐리고, 짚어놓고 바로 물러선다(*"…아닌가"*) |
| **비밀** | 그는 **경기를 계속 되감아 본다.** 재미가 아니라 습관이라 스스로 취미로도 안 친다. 남에게 말한 적이 없다 — 말할 가치가 있는 줄 몰랐으니까 |
| **머리에 박힌 말** | 부모의 ***"그걸로 뭐 먹고살래"*** — 회차 중 **한 번만** 인용된다(HANJI 자신의 입으로, 짧게). 부모는 화면에 없다 |
| **아크** | 명분을 구걸하러 온 아이 → 무심코 실력을 보임 → **이름을 받음** → *"프로는 안 될 수도 있어"*를 듣고도 **웃는다** |

### 핵심 대비 (이 캐릭터의 전부)
> **평소엔 흐린다. 경기 얘기를 할 때만 갑자기 또렷해진다.**

말끝이 사라지던 애가, 화면을 짚는 순간 **문장이 끝까지 간다.** 그리고 짚고 나서 **다시 흐려진다**(*"…그냥 보이는 대로 말한 건데요"*).
이 스위치가 켜졌다 꺼지는 것을 독자가 **귀로 듣게** 하라. 반전의 절반은 여기서 나온다.

### 목소리 시그니처
- **평소(흐림 모드)**: 짧은 문장. 말끝 생략 · 하강. `"아, 네…"` `"그냥… 좀 보고 있었어요."` `"…아닌가."`
  - 말버릇: 문장 앞에 **"아,"**를 붙이는 완충. 자기 말을 스스로 취소하는 꼬리 **"…아닌가요"**
  - 존댓말. RAKU에게도 존대 섞인 반말("어, 응… 그렇지").
- **분석 모드(또렷함)**: **문장이 끝난다.** 지시어가 생긴다. 어미가 단정형.
  - `"…아니, 그 전에요."` / `"여기 시야가 없었어요."` / `"이 타이밍에 이쪽을 못 봤어요."`
  - 이 모드의 문장에는 **"…"이 없다.** 레터러·작가는 이걸 규칙으로 지켜라.
- ⛔ HANJI는 자기를 낮추는 농담을 하지 않는다(자조 개그 금지 — 슬퍼지면 홍보툰이 아니다).

## 2. RAKU (라쿠) — 라쿤 · 중2 · **대비 주인공**

### 역할
반전의 저항선이자, 홍보 메시지의 **증명 사례**. 그가 수긍해야 이 회차가 팔린다.

| 항목 | 내용 |
|---|---|
| **신분·나이** | **중2.** 프로게이머 지망. **엄마가 등 떠밀어서** 왔다(본인은 굳이 필요 없다고 생각) |
| **want** | **인정.** "내가 잘한다"를 여기서도 확인받고 싶다 |
| **need** | **혼자서는 여기까지**라는 사실. 배우면 는다는 걸 **손으로** 겪는 것 |
| **결함** | 잘하니까 배우려 하지 않는다. 솔랭만 돈다. **패배의 원인을 항상 남/운/딜에 둔다** |
| **비밀** | 사실 **요즘 티어가 안 오른다.** 정체를 스스로 안다 — 그래서 더 크게 말한다. (대사로 자백하지 않는다. 지고 난 뒤 표정 R5에서 새어나온다) |
| **아크** | 깔봄 → 어이없음 → **패배** → 딱 하나 고침 → **결과가 달라짐** → 인정하기 싫은 머쓱함 → ***"…한 번 더 해볼게요."*** → 마지막에 **HANJI에게 묻는다** |

### ⛔ 하드룰 7 — **RAKU는 수긍한다**
끝까지 삐딱한 학생은 홍보툰에 마이너스다. 단, **완전히 굽히지 않는다** — 중2다.
수긍의 형태는 사과가 아니라 **행동**이다: 다시 잡는 마우스, 그리고 **질문**.

### 목소리 시그니처
- **반말 + 존댓말 뒤섞임**(어른한텐 존대, 형한텐 반말이 새어나옴). 문장이 짧고 빠르다. 어미가 튄다.
- 말버릇: 코웃음 **"치."** / 확신형 **"~잖아요"** / 남 탓 **"~때문이죠"**
  - `"한타 졌잖아요. 딜이 안 나와서."`
  - `"치, 말만 잘하네. 게임은 실력이지."`
  - `"…아, 진짜?"` (졌을 때, 화면에 대고)
- **HANJI 호칭 = "형"** — §4 온도 변화표 참조. **이 단어의 억양이 캐릭터 아크 그 자체다.**
- 수긍할 때도 사과하지 않는다: `"…한 번 더 해볼게요."` (눈은 화면에 둔 채)
- ⛔ 욕설·비속어 금지. 깐족은 **귀엽게**(홍보툰 · 청소년 독자).

## 3. HOO (후) — 부엉이 · **아카데미 멘토** (가상 마스코트)

### 역할
질문을 던지는 사람. **답을 주지 않고 되감는다.** 반전에 **이름을 붙이는** 유일한 인물.

| 항목 | 내용 |
|---|---|
| **신분** | **농심 이스포츠 아카데미의 멘토** — **가상의 마스코트다.** ⛔ 실존 강사 묘사·암시 금지 |
| **want** | 두 학생이 **자기 입으로** 답하게 만드는 것 |
| **need** | (그는 이 회차에서 변하지 않는다 — 촉매다.) 다만 **마지막의 짧은 미소**가 그의 유일한 크랙이다 |
| **결함** | 응원하지 않는다. 위로도 안 한다. 처음엔 차갑게 보인다 — 그래서 마지막 미소가 값이 있다 |
| **비밀** | 그는 **HANJI가 짚을 것을 알고 있었다**(1막에서 HANJI를 한 번 오래 본다 — 복선 1컷). 밝히지 않는다 |
| **기능** | RAKU에게는 **실력**을, HANJI에게는 **이름**을 준다. 같은 과제, **다른 답** |

### 목소리 시그니처
- **질문형이 기본.** 평서문보다 물음표가 많다. 짧다. 그러나 **차갑지 않다** — v1의 극단적 절제는 폐기됐다.
  - `"왜 졌는지 말해봐."` / `"너 방금 뭐 한 줄 알아?"` / `"들어가기 전에 여기 한 번 봐."`
- **따뜻하게 짚어준다**: 지적할 때도 상대를 부정하지 않고 **화면을 부정한다**(*"네가 틀린 게 아니라, 여기를 안 본 거야"*).
- ⛔ **응원 금지**: "할 수 있어" "힘내" "너 대단해" 류 금지. 대신 **사실을 말한다** — *"그게 전력분석관이 하는 일이야."*
- **현실을 숨기지 않는다**: *"프로는 안 될 수도 있어."* 그리고 곧바로 붙인다 — *"근데 네가 뭘 잘하는지는 여기서 알아볼 수 있어."*
- 반말(멘토 → 학생). 느리고 낮게. 문장 사이에 **한 박자**가 있다.
- ⛔ 직군명은 **일반 사실**로 말한다: *"이스포츠엔 이런 직업이 있어"* ○ / *"우리가 배출한다"* ✕ (사실 벽)

## 4. 관계망 — 이 회차의 진짜 플롯

```
            HOO (멘토 · 촉매)
           /                \
   "실력을 준다"          "이름을 준다"
        /                      \
     RAKU  ── "형" (온도가 바뀐다) ──▶  HANJI
    (손)                                (눈)
```

### ★★ 호칭 "형"의 온도표 (2차 반전 — **명시적으로 설계된 것이다**)

| 막 | 대사 | 온도 | 연기 지시 |
|:-:|---|---|---|
| **1** | *"형 고3이면… 좀 늦지 않았어요?"* | **❄ 깔봄** | RAKU=R2(위아래 스캔) / HANJI=H2(찔림). "형"에 **강세 없음** — 툭 던진다 |
| **2~3** | *"치, 말만 잘하네."* | **❄ 무시** | 여기서 RAKU는 HANJI를 **이름·호칭으로 부르지 않는다**(부재가 온도다) |
| **4** | (지고 난 뒤 — HANJI를 곁눈질) | **• 무언** | 대사 없음. R5 표정 + HANJI 쪽으로 눈만 |
| **5** | ***"형, 이거 어떻게 봐요?"*** | **🔥 배움** | RAKU=R9(정면·큰 눈, 몸이 돌아감) / HANJI=H9(살짝 웃음). "형"에 **강세 있음** — 진짜 부른다 |

> ⛔ **이 변화를 대사로 설명하지 마라.** 누구도 "쟤가 널 인정했어"라고 말하지 않는다.
> 하드룰 4(독자 추론 0) 위반이 아닌 이유: 이건 추론이 아니라 **감정의 인지**다(concept.md §6.2 승인).

### 쌍(pair)별 동력

| 쌍 | 처음 | 끝 | 회차 내 물리적 변화 |
|---|---|---|---|
| **RAKU → HANJI** | 깔봄(위에서 아래로 본다) | 배우려 함(**몸이 HANJI를 향한다**) | 의자 방향이 돌아간다 — 1막엔 등지고, 5막엔 마주 본다 |
| **HANJI → RAKU** | 위축(피한다) | 여유(마주 보고 웃는다) | 시선 회피 → **정면 응시** |
| **HOO → HANJI** | 관찰(O1) | 이름 부여(O6) → 현실 + 문 열어줌(O8) | 안경 반사광이 **걷힌다** |
| **HOO → RAKU** | 무판정(O3) | 한 개 고쳐줌(O7) → *"혼자 하면 딱 지금까지야"* | 팔짱 → 화면을 짚는 날개 |
| **HANJI ↔ 부모** | *"그걸로 뭐 먹고살래"*(1회 인용) | 회차 내 미해결 — **EP02의 씨앗** | ⛔ 부모 화면 등장 0 |
