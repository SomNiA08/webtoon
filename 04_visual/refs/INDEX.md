## 외형 벽 (위반 시 panel-validator C1 REGEN)

- **HANJI에게 안경을 씌우지 마라.** 원본 7장 중 1장(Hanji_5)에만 안경이 있었고 정본에서 제외했다. 안경은 HOO의 식별 표식이다.
- **HANJI에게 천사 날개를 달지 마라.** (원본 Hanji_6의 마스코트 변형 — 캐릭터 정본 아님)
- **헤드셋 색이 셋의 1차 식별자다**: HOO=네이비 / HANJI=청록 / RAKU=빨강. 섞이면 캐릭터가 무너진다.
- **눈 색**: HOO=호박 / HANJI=호박 / RAKU=**초록**(라쿤만 초록 — 대비 유지).
- 원본 이미지의 깨진 영문 글자("Aanjil", "Hanjo" 등)는 생성 아티팩트다. **패널에 전이 금지** — negative 프롬프트로 차단.
