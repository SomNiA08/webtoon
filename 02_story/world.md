# §0. 세계 개요 — 한 건물, 반나절

**무대는 단 하나: 농심 이스포츠 아카데미.** 대회장도, PC방도, 밤도 없다.
시간은 **어느 평일 오후, 체험 수업 한 타임**(대략 2~3시간). 창밖은 내내 **밝은 낮**이다.

이 세계는 판타지가 아니다. 규칙은 마법이 아니라 **"게임을 잘한다"의 정의**다.
- **RAKU의 세계**: 잘한다 = **손**이 빠른 것. 이기면 실력, 지면 팀 탓.
- **HANJI의 세계**: 잘한다 = **눈**으로 보이는 것. 근데 그걸 **아무도 값이라고 안 불러줬다.**
- **HOO의 세계(= 아카데미)**: **둘 다 게임이고, 둘 다 배울 수 있다.**

⛔ **어둡고 침울하게 만들지 마라.** 이 건물은 **밝고 따뜻하다.** 기본 조명 토큰은 style-bible §2-4의 **`WARM_DAY`**다.
어두워지는 곳은 **분석실 단 하나**뿐이며, 그마저 **`WARM_NIGHT`(코지한 남보라 + 앰버 램프)** — *칙칙함이 아니라 영화관의 어둠*이다.

# §1. 핵심 규칙 (대가 포함 — 이 5개가 이야기를 굴린다)

| # | 규칙 | **대가 / 제약** | 회수 지점 |
|:-:|---|---|---|
| **R1** | **화면은 거짓말을 안 한다.** 경기에서 벌어진 일은 리플레이에 **전부 남아 있다** | **되감을 줄 알아야 보인다.** 한 번만 보면 아무도 못 본다 — 그래서 "왜 졌냐"에 다들 *"딜이 안 나와서"*라고 답한다 | 막2 — HOO가 되감고, HANJI가 짚은 지점에서 **정말로 무너진다** |
| **R2** | ★★★ **안 보이는 곳에서 진다.** 패배는 한타가 아니라, **한타 전에 시야를 잃은 순간** 결정돼 있다 | **시야를 확보하려면 캐리를 포기해야 한다** — 눈에 안 띄는 일이라 아무도 칭찬 안 한다 | 이 회차 전체의 물증. §3 참조 |
| **R3** | **무심코 한 것에는 이름이 없다.** 이름이 붙는 순간, 그것은 **취미에서 직업이 된다** | **이름은 혼자서는 못 붙인다.** 누군가 그것을 **보고 불러줘야** 한다 → *그래서 아카데미가 필요하다* | ★★★ 1차 반전 — *"그게 전력분석관이 하는 일이야."* |
| **R4** | **혼자 하면 딱 지금까지다.** 솔로 랭크는 실력을 **측정**하지만 **가르치지는 않는다** | **배우려면 남에게 진 이유를 들려줘야 한다** — 중2에겐 그게 제일 어렵다 | 막4 — RAKU가 한 판 지고, **딱 하나** 고쳐서 다시 이긴다 |
| **R5** | **가르침은 한 번에 하나만.** HOO는 **딱 하나만** 고친다(*"들어가기 전에 여기 한 번 봐"*) | 열 개를 고치면 **아무것도 안 바뀐다.** 하나를 고쳐야 **결과가 바뀐 이유를 안다** | 막4 — 변수가 하나라서 **증명이 성립한다** |

> **R2 + R3이 이 회차의 명제다.** 안 보이는 곳을 보는 사람이 있고(R2), 그 사람에게 **직업의 이름이 있다**(R3).

# §2. 공간 — 신규 LOC 5종 (art-director: style-bible §6 전면 교체)

> **v1 LOC 7종(LOC_STAGE · LOC_BOOTH · LOC_PCBANG · LOC_BACKSTAGE · 구 LOC_PRACTICE · 구 LOC_DESK · 구 LOC_CORRIDOR) 전부 폐기.**
> 아래 5종이 이 회차의 전부다. 동선: **로비 → 복도 → 연습실 → 복도 → 분석실 → 연습실 → 로비**.
> 각 LOC의 문자열은 **art-director가 style-bible §6 문법(잉크 선 · 둥근 소품 · 셀 채색)으로 다듬어 확정**한다. 아래는 **무엇이 있어야 하는가**의 정본이다.

## LOC_LOBBY — 아카데미 입구 · 로비 (첫 컷 / 마지막 컷) · `WARM_DAY`

- **조도**: **가장 밝다.** 통유리로 들어오는 **골든 오후 햇살**, 크림 화이트 벽, 따뜻한 나무 바닥. **낮 최고 조도.**
- ★ **고정 소품(연속성 앵커)**:
  1. **벽 명패 — `농심 이스포츠 아카데미`** (한글, 굵은 고딕, **판독 가능**. brief 하드룰 6 = 확정 사실). ⛔ 명패에 **다른 문구·영문·수강료·실적 0**.
  2. **체험 수업 안내 스탠드 배너** — **글자 없는 빈 배너**(`[[TBD: 아카데미 정보]]`). ⛔ 커리큘럼·기간·요금 **지어내기 금지**.
  3. **둥근 소파 하나 + 낮은 테이블** — RAKU가 발 구르며 앉아 있는 곳(막1 도입).
  4. **화분 하나**(둥근 잎). 이 건물이 **차갑지 않다**는 증거.
- **연출 규칙**: **명패는 첫 등장 컷과 엔드카드 직전 컷, 두 번만 정면으로 보인다.** 나머지는 프레임 밖.
- **대구 규약**: **P(초반) 로비 = 둘이 서로 모른 척** ↔ **P(막5) 로비 = RAKU가 HANJI에게 묻는다.** 같은 소파, 같은 햇살, 다른 거리.

## LOC_CORRIDOR — 아카데미 복도 (이동·호흡) · `WARM_DAY`

- **조도**: 밝다. 한쪽 벽이 **창(블라인드 반쯤)** — **골든 슬랫 빛이 바닥에 줄무늬**를 긋는다.
- ★ **고정 소품**:
  1. **바닥의 빛 줄무늬**(블라인드 그림자) — ★ **대구 앵커**: 막1 HANJI는 **줄무늬를 밟지 않고 벽 쪽으로 붙어 지나간다** / 막5 HANJI는 **줄무늬 한가운데 선다.** 같은 복도, 같은 빛, 다른 걸음.
  2. **문 3개**(연습실 · 분석실 · 빈 문 하나). **문에 붙은 표찰은 빈 슬롯**(글자 0 — 로비 명패 외 기관 고유물 0).
  3. **정수기 + 종이컵** — HOO가 컵 들고 서 있는 컷.
- ⛔ **복도를 어둡게 만들지 마라.** v1의 밤 복도는 폐기됐다.

## LOC_PRACTICE — 연습실 (두 학생이 만나는 곳 · 막1 · 막4) · `WARM_DAY`

- **조도**: **크림 웜 화이트 천장등 + 블라인드 골든 햇살.** ⛔ 차가운 형광등·회색 사무실 금지(v1 P26 실패 원인).
- ★ **고정 소품**:
  1. **5연석 PC 좌석 열**(정면 나란히) + **벽쪽 1인석 1개**. → **RAKU = 1인석**(혼자 솔랭 = R4의 시각화) / **HANJI = 5연석 끝자리**.
  2. **둥근 게이밍 체어**(치비 몸에 맞는 스케일 — 등받이가 머리 위로).
  3. **바닥 매트의 바퀴 자국** — ★ **방향이 증거다: RAKU의 1인석에서 5연석 쪽으로.** ⛔ **세지 마라**(세면 숫자가 된다).
  4. **빈 화이트보드** — 격자만. ⛔ 커리큘럼·글자 0.
  5. **음료 캔 두 개** — 막1엔 따로 떨어져 있고, 막5엔 나란히 놓여 있다(대사 0의 조용한 앵커, **클로즈업 금지 — 배경에만**).
- **연출 규칙**: 막1의 두 사람 사이 **거리**(좌석 3칸)와 막4의 **거리**(옆자리)가 관계 변화를 말한다.

## LOC_ANALYSIS — ★★★ 분석실 (반전의 무대 · 막2·막3) · `WARM_NIGHT` (단, **아늑한 어둠**)

> **이 회차 유일하게 조도가 내려가는 방.** 이유는 이야기 장치다 — **불을 끄면 화면이 더 잘 보인다**(R1).
> ⛔ **침울함이 아니다.** 남보라 어둠 + **앰버 스탠드 1개 필수** + 청록 화면광. **영화관의 어둠, 상영 전의 설렘.**

- **조도**: 천장등 **끔** · 블라인드 **내림**(가장자리에서 **골든 낮빛이 새어 들어온다** — 여긴 아직 낮이라는 증거이자 따뜻함의 보증).
- ★ **고정 소품**:
  1. ★★★ **벽면 대형 화면 1장**(프로젝션/대형 모니터) — **리플레이가 재생되는 곳. 이 회차의 물증 화면.** (§3 전문)
  2. ★★ **직군 소개 보드**(벽면) — §4 전문. **하드룰 5의 집행 장치.**
  3. **앰버 데스크 램프 1개** — style-bible §8-0-6(어두운 씬 = 따뜻한 광원 필수) 준수.
  4. **긴 회의 테이블 + 둥근 의자 3개** — HOO · HANJI · RAKU. **막2엔 셋이 떨어져 앉고, 막3 끝엔 HANJI가 화면 앞으로 나와 있다.**
  5. **레이저 포인터 대신 — HOO의 앞발(손)**. ⛔ 포인터·화살표·강조 원 **금지**: **짚는 것은 손가락이다**(HANJI의 짚는 손 = 이 회차의 결정적 제스처).
  6. **종이컵 물 2개**(깨끗한 종이컵 — 얼룩·때 금지).
- **연출 규칙**: **화면광(청록)이 세 얼굴에 다르게 얹힌다** — RAKU는 눈부셔하고, HANJI는 **몸을 앞으로 기울인다**, HOO는 **아이들의 얼굴을 본다**(화면을 안 본다).
- ★ **HANJI의 짚는 손 = 이 회차 최고 앵커.** 짚는 컷은 **얼굴과 손이 같은 프레임**에 들어와야 한다(하드룰 1 — 손만 나오는 클로즈업 예산을 여기 쓰지 마라).

## LOC_ENDCARD — 엔드카드 (2장 · 이미 확정) · `WARM_DAY`

- **로비 명패**(`농심 이스포츠 아카데미`) + **개설 과정 3종**(리그 오브 레전드 · 발로란트 · FC 온라인 — **확정 사실, 표기 가능**) + **빈 정사각 QR 슬롯**.
- ⚠️ **QR은 렌더하지 않는다** — 빈 정사각 슬롯만 렌더하고 `00_input/academy_blog_qr.jpg`를 **PIL로 픽셀 합성**(v1 검증 기법).
- ⛔ 수강료·기간·취업률·실적 **0**.

# §3. ★★★ 게임 화면 규칙 (반전의 물증 — 이 회차에서 가장 중요한 시각 장치)

## 3-0. 저작권 벽 (brief 하드룰 8)

⛔ **실존 게임(LoL/발로란트/FC온라인)의 UI · 챔피언 · 스킬 아이콘 · HUD 레이아웃 · 로고 복제 절대 금지.**
✅ **일반화된 5v5 게임**으로 추상화한다 — 팀은 **파랑 vs 빨강**, 캐릭터는 **동그란 점/실루엣**, 맵은 **추상 도형**.

> **버리는 것**: 실존 게임의 그래픽.
> **반드시 남기는 것 4개**: ⓐ **미니맵** ⓑ **시야(안 보이는 영역)** ⓒ **리플레이 타임라인 + 되감기** ⓓ **한타(5명이 한 곳에 뭉쳐 싸운다)**.
> 이 4개가 없으면 **반전이 성립하지 않는다.**

## 3-1. 화면의 3층 구조 (분석실 대형 화면 = 항상 이 구성)

```
┌──────────────────────────────────────────────────────────────┐
│  ① 경기 뷰 (화면의 대부분)                                    │
│     위에서 내려다본 추상 전장. 파란 원 5개 · 빨간 원 5개.      │
│     바닥은 짙은 청록 격자. 지형은 단순한 둥근 덩어리.          │
│                                                              │
│                                    ┌───────────────┐         │
│                                    │ ② 미니맵      │ ← 우하단 │
│                                    │  둥근 사각형   │   고정   │
│                                    └───────────────┘         │
├──────────────────────────────────────────────────────────────┤
│  ③ 리플레이 타임라인 바 (화면 하단 가로 전체)                  │
│     ◀◀ [되감기]  ━━━━━━━●━━━━━━━━━━━━━━  (스크러버)          │
└──────────────────────────────────────────────────────────────┘
```

| 층 | 명세 |
|:-:|---|
| **① 경기 뷰** | **탑다운 추상 전장.** 아군 = **파란 원 5개**(둥근 점, 흰 테두리) / 적 = **빨간 원 5개**. 지형 = **단순한 둥근 덩어리 + 청록 선의 길**. ⛔ 챔피언 초상·스킬바·체력 숫자 **0** |
| **② 미니맵** | **우하단 고정.** 둥근 모서리 정사각형. 전장 전체의 축소판. **여기에 시야가 그려진다** — §3-2 |
| **③ 타임라인** | **하단 가로 바.** 왼쪽에 **`◀◀` 되감기 아이콘 + 한글 라벨 `되감기`**. 바 위를 **동그란 스크러버**가 달린다. HOO의 손이 **스크러버를 뒤로 끈다** |

## 3-2. ★★★ "시야가 없다"를 어떻게 그리는가 (이 회차 최대 시각 장치)

> **정답 = 미니맵 위에 덮인 어두운 덮개 = `FOG_PATCH`.**

```
   미니맵 (둥근 사각형)
 ┌───────────────────────┐
 │▓▓▓▓▓▓▓▓▓╲             │  ← ★ FOG_PATCH: 왼쪽 위 구역이
 │▓▓▓▓▓▓▓▓▓▓╲            │     짙은 남보라 반투명 덮개로 덮여 있다.
 │▓▓▓▓▓▓▓▓╱              │     가장자리는 부드럽고 불규칙한 곡선.
 │ ╲▓▓▓▓╱      ●  ●      │     ⛔ 그 안에 아무것도 안 보인다 — 빨간 점 0.
 │          ●  ●  ●      │  ← 파란 점 5개는 반대편(오른쪽 아래)에 뭉쳐 있다
 └───────────────────────┘
```

| 축 | 명세 (prompt-smith · panel-validator 판정 기준) |
|---|---|
| **형태** | 미니맵의 **한 구역(왼쪽 위 사분면)**이 **짙은 남보라 반투명 덮개**로 덮인다. 가장자리 = **부드러운 곡선**(직선 아님, 사진 노이즈 아님) |
| **색** | 덮개 = **짙은 남보라 `#2E2A4F`계 반투명** (⛔ **검정 금지** — style-bible §8-0-5). 밝은 구역 = **청록 격자가 또렷이 보인다** |
| **대비** | ★ **덮인 구역과 안 덮인 구역이 한눈에 갈린다.** 어중간한 그라데이션 금지 — **경계가 읽혀야 한다** |
| ★ **라벨** | **덮개 위(또는 옆)에 짧은 한글 라벨 하나** — **`시야 없음`** (4자, 굵은 고딕, 흰색 또는 청록). ★★★ **brief 하드룰 4의 집행**: 독자가 "저 어두운 게 뭐지?"를 **추론할 필요가 없다** |
| ★ **대사와 이중 결속** | 라벨만으로 끝내지 않는다 — **HANJI가 손가락으로 그 구역을 짚으며 말한다**: *"여기 시야가 없었어요."* **화면 라벨 + 캐릭터 대사 + 짚는 손 = 3중 전달.** 이것이 v1이 실패한 지점의 정확한 반대다 |
| **점의 배치** | 덮개 안 = **아무 점도 없다**(그래서 무섭다) / 파란 점 5개 = **반대편에 뭉쳐 있다**(적을 안 보고 있다) |

### ★ 시야 장치의 3단 회수 (막2~막3 — 반드시 이 순서)

| 단계 | 화면 | 대사(하드룰 4 — 화면에 반드시 말이 붙는다) |
|:-:|---|---|
| **1. 짚는다** | 미니맵 **정지 화면**. 왼쪽 위 `FOG_PATCH` + `시야 없음` 라벨 | HANJI: *"…아니, 그 전에요. **여기** 시야가 없었어요."* (짚는 손 + 얼굴 같은 프레임) |
| **2. 되감는다** | HOO의 손이 **스크러버를 뒤로 끈다.** 타임라인의 `되감기` 라벨 + `◀◀` | HOO: *"되감아 볼까."* (짧게 — 화면 라벨과 자구가 겹쳐도 좋다) |
| **3. ★ 터진다** | ★★★ **그 어두운 구역에서 빨간 원 5개가 쏟아져 나온다.** 파란 원 4개를 **둘러싼다** → **한타 패배.** 덮개는 여전히 거기 있다 | RAKU: *"…어?"* / HOO: *"봤지? 여기서 진 거야."* — **독자가 추론할 것 = 0** |

> ★★★ **3단계가 이 회차의 심장이다.** *"안 보이는 곳에서 적이 나온다"* — 이 한 장면이 R2를 **말 없이도 증명하고, 말로도 못 박는다.**
> panel-validator: **★SSOT 앵커.** ACCEPT-FLAG 통과 불가 — 기준 만족까지 재렌더.

## 3-3. 화면 안 한글 라벨 — **화이트리스트 (이것만 쓴다)**

> ✅ **brief 하드룰 4 = 화면 증거에 라벨을 붙여도 된다.** v1의 "화면 문자 0" 룰은 **폐기됐다.**
> ⚠️ 단 **짧게 · 굵은 고딕 · 최대 4자** — 길면 codex가 글자를 깨뜨린다.

| 라벨 | 글자수 | 어디에 |
|---|:-:|---|
| **`시야 없음`** | 4 | ★ 미니맵 `FOG_PATCH` 위 |
| **`되감기`** | 3 | 타임라인 좌측 `◀◀` 옆 |
| **`한타`** | 2 | (선택) 한타 순간의 화면 하단 작은 태그 |
| **`전력분석관` `코치` `감독` `운영`** | 3~5 | ★ 직군 보드(§4) — **화면이 아니라 벽 보드** |
| **`농심 이스포츠 아카데미`** | — | 로비 명패 · 엔드카드 **한정** |
| **`리그 오브 레전드` `발로란트` `FC 온라인`** | — | **엔드카드 한정**(확정 사실) |

⛔ **위 목록 밖의 화면 문자는 전부 금지.**
⛔ **판독 가능한 숫자 0** — KDA · 체력 · 시간 · 점수 · 승률 **전부 금지**. 눈금이 필요하면 **숫자 없는 짧은 틱 마크**만.
⛔ **영문 0.** ⛔ **깨진 한글 = 즉시 REGEN.**

⛔ **스캔라인 · 글리치 · 픽셀 노이즈 · 사진 화면 질감 · 렌즈 플레어 0.**

# §8. 시각 톤 (art-director · prompt-smith)

## 8-1. 조도의 위계 (v1의 4단계 폐기 → 신규 3단계)

```
[가장 밝음]  LOC_LOBBY      통유리 골든 햇살 · 크림 화이트     ← 들어오는 곳 · 나가는 곳
[밝음]       LOC_CORRIDOR   블라인드 줄무늬 빛                ← 이동
[밝음]       LOC_PRACTICE   크림 웜 화이트 + 골든 햇살         ← 손이 배우는 곳
[아늑한 어둠] LOC_ANALYSIS   남보라 + 앰버 램프 + 청록 화면광    ← ★ 눈이 배우는 곳 · 반전
```

> **불을 끄는 방이 이 회차에서 유일하게 진실이 보이는 방이다** (R1).
> ⚠️ **어둠 ≠ 침울.** 분석실은 **영화관**이지 심문실이 아니다. **앰버 램프 1개 · 블라인드 틈의 낮빛 · 세 얼굴에 얹힌 따뜻한 림라이트** 필수.

## 8-2. 색

- **화면광(청록 `#4FC3F7`~`#2FB3A8`)** = **차가운 키** — 분석실 세 얼굴의 한쪽.
- **앰버(`#FFB870`)** = **따뜻한 키** — 반대쪽. **모든 어두운 컷에 필수.**
- **그림자 = 인디고-보라(`#4A3F6B`)** — ⛔ **검정·중성 회색 금지.**
- **낮 실내 = 크림 웜 화이트 + 골든 블라인드 빛 + 피치 톤 그림자.**
- **헤드셋 색 = 1차 식별자**: HOO **네이비** / HANJI **시안청록** / RAKU **빨강**. 눈: **RAKU만 초록.**
- ★ **청록 서명 규약 유지**: **HANJI의 청록**과 **화면의 청록**은 같은 계열이다 — 그가 짚을 때 **손과 화면이 같은 색으로 만난다.** ⛔ RAKU·HOO 소품에 청록 금지.

## 8-3. 연기 (★ v1 폐기 룰의 반대)

> ⛔ ***"과장 리액션 금지" · "표정 변화는 작게" 룰은 폐기됐다.***
> ✅ **마스코트의 표정 매력을 적극적으로 쓴다.**

| 캐릭터 | 이 회차의 표정 |
|---|---|
| **RAKU** | 깔봄(눈 반쯤) → **억울함**(입 삐죽) → **놀람**(눈 커짐, 귀 쫑긋) → **머쓱함**(뺨 긁적) → **호기심**(몸 앞으로) |
| **HANJI** | 위축(어깨 움츠림) → **머뭇거림** → ★ **자기도 모르게 확신에 찬 얼굴**(짚는 순간) → **멍함**(이름을 들었을 때) → **뿌듯함**(작게, 마지막) |
| **HOO** | 잔잔한 미소 · **눈을 가늘게 뜨고 아이를 본다**(화면이 아니라 얼굴을 본다) · 딱 한 번 **크게 웃는다** |

- **얼굴 보이는 패널 ≥ 70%** (하드룰 1). **무대사 ≤ 8장** (하드룰 2).
- **효과선은 절제하되 금지는 아니다** — 놀람의 순간(한타가 터질 때)엔 **가벼운 임팩트 표현 허용.** ⛔ 글리치·스캔라인은 여전히 금지.
